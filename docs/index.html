How it works

Three keys. That's the whole app.

1
It's always recording in the background — a rolling buffer of your last few seconds, kept in memory.
2
Made a play? Hit ⌥⌘C to save the last 30 seconds, numbered cleanly as CLIP NO. 1.mp4.
3
Hit ⌥⌘S to upload it and copy a Discord-playable link straight to your clipboard.

Install in one command

The tap clears the macOS quarantine flag for you, so it just opens.

# add the tap, then install brew tap Bhavyaapple24-ai/clipthat brew install --cask clipthat

On first launch, macOS asks for Screen & System Audio Recording permission — grant it to “ClipThat,” then reopen. Prefer to build it yourself? Clone the repo and run ./scripts/bundle.sh.

Questions

Is it actually free?

Yes — free and open source under the MIT license. No account, no trial, no telemetry. You can read every line of the source on GitHub.

Why does it need screen recording permission?

That's how it captures your gameplay — the same permission OBS or any screen recorder uses. Audio capture rides on the same grant. Nothing leaves your Mac unless you press the share key.

Will it slow down my game?

No. Encoding runs on the dedicated hardware encoder, and only the compressed video is kept in memory, so there's no measurable FPS hit while playing.

Does it work on Intel Macs?

It's built and tested for Apple Silicon on macOS 15 (Sequoia) and newer. Intel isn't officially supported.
